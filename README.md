## Описание программы
** «GitHub User Finder» - Приложение для поиска пользователей GitHub с использованием внешнего API и сохранением избранных пользователей и Git.**
## Требования для запуска
- Python 3.10 или выше
- Библиотеки: 'pip install requests', `tkinter` — для графического интерфейса
## Как получить API-ключ
Для поиска используется GET-запрос: https://api.github.com/users/{username}
## Примеры использования
1. Команда запуска `python main.py`
 2. Введите логин пользователя GitHub и нажмите "Поиск".
3. Просмотрите информацию о пользователе.
4. Нажмите "Добавить в избранное" для сохранения пользователя.
5. Избранные пользователи сохраняются в файл favorites.json.
